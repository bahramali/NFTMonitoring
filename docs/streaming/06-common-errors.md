# خطاهای رایج

این بخش برای خطایابی سریع است.

## 1) خطای 404 روی `index.m3u8`

- مسیر دوربین در MediaMTX درست نیست.
- URL اشتباه است یا پورت در نظر گرفته نشده.

## 2) CORS Error

- هدرهای CORS در Caddy یا MediaMTX تنظیم نشده.
- درخواست از دامنهٔ متفاوت می‌آید و مرورگر اجازه نمی‌دهد.

## 3) TLS / Certificate Error

- گواهی منقضی یا دامنه اشتباه است.
- Cloudflare روی حالت ناسازگار TLS است.

## 4) قطعات `.ts` لود نمی‌شوند

- احتمالاً مسیر پورت یا پروکسی اشتباه است.
- ممکن است Cloudflare یا فایروال جلوی درخواست را بگیرد.

## 5) تأخیر زیاد

- segment duration زیاد است.
- playlist size بزرگ است.

## 6) تصویر سیاه یا لگ

- نرخ بیت بیش از ظرفیت شبکه است.
- قطع و وصل شدن RTSP در منبع.
