# MediaMTX

MediaMTX قلب سیستم استریمینگ است و وظیفهٔ دریافت RTSP و خروجی دادن HLS را دارد.

## وظایف اصلی

- دریافت استریم RTSP از دوربین.
- ساخت HLS با خروجی `index.m3u8` و قطعات `.ts`.
- مدیریت مسیرهای چند دوربین.

## ساختار مسیرها

هر دوربین در MediaMTX یک مسیر (path) دارد. خروجی HLS معمولاً در مسیر زیر است:

```
http://<MEDIAMTX_HOST>:<PORT>/<CAMERA_ID>/index.m3u8
```

## نکات عملیاتی

- نام مسیر باید دقیقاً با شناسهٔ دوربین یکی باشد.
- اگر stream قطع شد، MediaMTX به‌صورت خودکار تلاش برای reconnect می‌کند (بسته به تنظیمات).
- بهتر است مسیرهای RTSP را ثابت و قابل اطمینان نگه دارید.

## تنظیمات نمونه (مفهومی)

- تعریف مسیر برای هر دوربین.
- فعال بودن خروجی HLS.
- تنظیمات segment duration و playlist size.

> تنظیمات دقیق بسته به فایل پیکربندی شما است، اما ساختار کلی در همهٔ پروژه‌ها مشابه است.
